# Dashboard-Power-BI-Torre-de-Controle-Log-stica

O objetivo desenvolvido nesse Dashboard foi criar uma torre de controle logística para monitorar e melhorar o desempenho das operações de transporte, centralizando dados logísticos em um único painel para apoiar decisões, aumentar a eficiência operacional e melhorar o nível de serviço (entregas noprazo e sem falhas).

Principais elementos exibidos:

* Acompanhamento de desempenho ao longo do tempo -> análise de tendências operacionais
* Indicador de OTIF (on time in full) -> medir a qualidade das entregas
* OTD por transportadora (on time delivery) -> compara eficiência entre transportadoras
* Top clientes e volumes faturados -> foco em receita e priorização de clientes
* Desvios operacionais -> identificar problemas como avarias e inconsitências
* Mapa geográfico -> visualizar a distribuição das operações por região
* Filtros de transporte (tipo de transporte, ano, origem) -> permitir análise segmentada
